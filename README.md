# qr-code
## Install command for termux
Install python and pip, ⇓

```bash
  pkg install python3
```
```bash
  pkg install git
```
install the tool ⇓

```bash
  https://github.com/Halal-AsHacker/qr-code.git
```
Now ⇓

```bash
  cd qr-code/ 
```

 Give all permission ⇓

```bash
  chmod 777 requiremen.sh
```
Install the all requirement ⇓

```bash
  ./requiremen.sh
```
At last ,Now  you can run the tool in 2 way ⇓

```bash
  python qrcode.py
```
